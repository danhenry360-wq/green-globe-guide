# BudQuest - Global Cannabis Travel Guide

Your ultimate guide to 420-friendly destinations worldwide.

**Live URL**: https://budquest.guide

## How to Edit

### Using Your Preferred IDE

Clone the repo and push changes:

```sh
# Clone the repository
git clone https://github.com/danhenry360-wq/green-globe-guide.git

# Navigate to the project directory
cd green-globe-guide

# Install dependencies
npm install

# Start the development server
npm run dev
```

### Edit Directly in GitHub

- Navigate to the desired file(s)
- Click the "Edit" button (pencil icon)
- Make your changes and commit
